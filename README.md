# DVSA Proxy Backend - London Slot Monitor

## Deploy on Railway.app

1. Fork this repo
2. Go to railway.app
3. New Project → Deploy from GitHub
4. Select this repo
5. Deploy

## First Time Setup

1. Login to DVSA in browser
2. F12 → Console → `document.cookie` → copy
3. Open your Railway URL
4. Click "Set Cookie" → paste → submit
5. Click "Scan All Centres"

## Daily Use

- Keep DVSA logged in browser
- Re-set cookie if session expires
- Click "Scan All Centres" to check slots
